Project title

The smart bathtub is a monitoring system that manages the water temperature in a bath. This system utilizes the Raspberry Pi and Arduino along with two water pumps, two relays, a temperature sensor, and two liquid flow meters. This system was created to provide parents with a convenient and reliable alternative for bath time.

Sets of code were created using Python on the Raspberry Pi that would allow the user to input the volume of the bathtub, the desired water temperature, and initial water temperature. The Raspberry Pi would integrate the programmed information and run parameters that allow the 4 channel 5V Relay to disengage both the hot water pump and the cold water pump depending on the desired temperature.

Project title

To test the effectiveness of different antiperspirants, a Raspberry pi with a humidity sensor was used to measure the humidity created after a workout. Two Python programs were created: one that used the humidity sensor on the Sense HAT to collect the data from the sensor and the other to import that data into a Comma Separated Value (.csv) file for analyzation. To power and make the Raspberry Pi 3 portable so data collection could be immediate, a 16000 milliamp USB battery pack was used.
